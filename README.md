 # NoSQL Databases Analysis

## Overview

This project presents a research-based analysis of NoSQL databases and their role in modern data systems.

It explores how traditional relational databases (RDBMS) struggle with scalability and how NoSQL solutions address the needs of large-scale, distributed and high-performance applications.

---

## Key Topics

* Evolution from relational databases to NoSQL
* Distributed systems and horizontal scalability
* ACID vs BASE consistency models
* Types of NoSQL databases:

  * Document-oriented
  * Key-value stores
  * Column-family databases
  * Graph databases
* Advantages and limitations of NoSQL
* Real-world applications in industries such as e-commerce, social media and finance
